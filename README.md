# IdeaZen - AI Project Idea Generator

This is a React application built with Vite, TypeScript, and Tailwind CSS.

## Prerequisites

You need **Node.js** installed on your computer.
Download it here: [https://nodejs.org/](https://nodejs.org/)

## How to Run

1.  Open a terminal (Command Prompt, PowerShell, or VS Code terminal).
2.  Navigate to this project folder:
    ```bash
    cd c:\One_For_All\Ideazen
    ```
3.  Install the dependencies:
    ```bash
    npm install
    ```
4.  Start the development server:
    ```bash
    npm run dev
    ```
5.  Open the link shown in the terminal (usually `http://localhost:5173`) in your browser.

## Features

*   **Landing Page**: Premium UI with animations.
*   **Skill Assessment**: Interactive cards to select your level.
*   **Question Flow**: Dynamic questions based on interests and time.
*   **AI Output**: Detailed project proposals with difficulty scores and roadmaps.
